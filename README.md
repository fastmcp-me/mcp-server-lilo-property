# lilo MCP Server

<!-- mcp-name: io.github.lilo-property/mcp-server -->

Vacation rental booking and protection for AI agents.

## Quick Start

Get your API key instantly:

```bash
curl https://mcp.lilo.property/developers/key
```

## MCP Endpoint

```
https://mcp.lilo.property/mcp
```

## Available Tools

| Tool | Description | Credits |
|------|-------------|---------|
| search_properties | Find vacation rentals by location | 1 |
| check_availability | Real-time pricing and dates | 1 |
| create_booking | Book with instant confirmation | 2 |
| check_guest_risk_score | Pre-booking risk assessment | 5 |
| detect_extortion_pattern | AI threat detection | 10 |
| get_squatter_risk | 50-state tenancy analysis | 3 |
